## App create

1. first step, you have to build project

```terminal
yarn build
```

2. copy vue file to ios/android folder
```terminal
npx cap copy
```

3. open app IDE
```terminal
npx cap open ios
npx cap open android
```

4. finish create and you can open app from IDE
